Principais Formatações (tags semânticas)

Negrito / Destaque

Nesta frase temos um termo em destaque usando a tag <strong>

Itálico / Ênfase

Nesta frase, temos um termo em ênfase usando a tag <em>

Texto Marcado

Podemos criar um texto marcado com a tag <mark>

Texto Grande e Pequeno

Texto Pequeno com a tag <small> O texto grande é feito com CSS por conta da semântica

Texto Deletado

Podemos marcar um texto como excluído com a tag <del> para indicar que ele deve ser lido mas não considerado

Texto Inserido

Podemos marcar um texto como inserido com a tag <ins> para indicar que ele foi adicionado depois

Texto Sobrescrito

Texto sobrescrito como X2 utilizamos a tag <sup>

Texto Subescrito

Texto subescrito como H2O utilizamos a tag <sub>
